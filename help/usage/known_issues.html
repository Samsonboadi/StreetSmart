Known Issues¶

  • If you select a photo from the timeline in the viewer, the Recordings WFS will continue to show the current recordings instead of the recordings at the chosen timeline.

  • After installing the plugin, a restart is required to see the plugin buttons in the toolbar.

  • After a fresh installation of the plugin, you need to restart QGIS for the Panoramaviewer to work. Otherwise, the Panoramaviewer will keep loading, and the images will not display when you select a cyclorama point.

  • If you add a new measurement, it will only be displayed as an overlay if you click the Add Overlay button again or if you select a subsequent photo in the viewer.
